# API Testing Sandbox

Практика тестирования API (REST) и связанных SQL-запросов.

## В проекте:
- **Примеры из Postman:** Скриншоты успешных запросов.
- **SQL-запросы:** Примеры проверок данных в БД.
- **Тест-кейсы:** Структурированные сценарии.

## Инструменты:
- Postman, JSONPlaceholder API
- PostgreSQL, DBeaver
- Git, GitHub

## Цель:
Демонстрация базового workflow тестирования API и проверки целостности данных.
